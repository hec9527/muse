# Muse publish tool

`muse` 工具是用于 `bid` 所构建项目的发布工具，区别于 `bid` 的命令行模式， `muse` 基于 `vscode` 扩展提供了 `GUI` 操作界面，更加简单高效，告别了繁琐的输入选择发布的繁琐流程，让代码发布以及重新发布变得更加 `human friendly`。同时也提供了更加实用的特性

## 🛠 feture

- 缓存用户信息

- 基于图形化的操作界面

- 快速过滤选择页面

- 新增大量快捷键

- 保存发布记录，下次快速选择发布内容，节省发布时间

- 发布日志结果监听，失败后一键重新发布
